Student Management System

Student Management System is a Spring Boot web application designed as a learning project to strengthen backend development fundamentals. It demonstrates CRUD operations and integrates MySQL database with Hibernate/JPA, along with a responsive frontend using Thymeleaf, HTML, CSS, and Bootstrap.


Features

Add, view, update, and delete student records (CRUD operations)
Responsive UI built with Bootstrap and Thymeleaf
Clear separation of concerns using MVC architecture
Integration with MySQL using Hibernate/JPA


Tech Stack

Backend: Java, Spring Boot (Maven), Spring Web (MVC), Hibernate, Spring Data JPA
Database: MySQL
Frontend: Thymeleaf, HTML, CSS, Bootstrap, JavaScript


Key Learnings

Implementing MVC architecture in Spring Boot
Handling HTTP requests via @GetMapping and @PostMapping
Connecting Spring Boot with MySQL using Hibernate/JPA
Designing a responsive frontend with Thymeleaf + Bootstrap
Applying OOP concepts such as classes, interfaces, and separation of concerns
Understanding end-to-end application flow from request to database to view



How to Run

Clone the repository:

git clone https://github.com/<username>/StudentManagementSystem.git


Import the project in IntelliJ IDEA or any IDE

Configure MySQL database and update application.properties

Run the application using:

mvn spring-boot:run


Open browser: http://localhost:8080/students

Project Goal

This project is a hands-on learning application to connect theory with practice in Spring Boot backend development. It helped in understanding controllers, database integration, CRUD operations, and MVC flow.
